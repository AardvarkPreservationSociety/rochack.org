IRC: Internet Relay Chat

RocHack has an IRC channel. We use it for talking about things and stuff.

Server: irc.freenode.net
Channel: ##rochack

What is IRC?

IRC is a text-based chat protocol. It is used for people to communicate and is popular in open source developer communities.

How do I connect?

There are many clients to choose from. Here are some good ones. If there is one not on this list that you use and recommend, feel free to make a pull request adding it.

Command line

  • irssi
  • weechat
  • finch

Web

  • IRCCloud
  • Kiwi IRC

Cross-platform

  • HexChat

    • Mac OS X

    • LimeChat
    • Textual
    • Adium
    • Linkinus

    iOS

    • Colloquy
    • Palaver
